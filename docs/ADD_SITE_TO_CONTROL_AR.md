# ➕ إضافة موقع للمراقبة

## الخطوات السريعة

1. سجّل دخول كمدير → `/admin/control-center`
2. اضغط **"إضافة موقع"**
3. املأ الاسم + الدومين (إجباريان) — الباقي اختياري
4. اضغط **"إضافة الموقع"**

## الحقول

| الحقل | مطلوب | مثال |
|---|---|---|
| الاسم | ✅ | غسيل السيارات |
| الدومين | ✅ | `example.com` (بدون https://) |
| GitHub Repo | ❌ | `https://github.com/user/repo` |
| الفرع | ❌ | `main` |
| السيرفر الأصلي | ❌ | LWS / Vercel |
| ملاحظات | ❌ | حر |

## ماذا يحدث بعد الإضافة؟

- الموقع يدخل دورة الفحص التلقائي (كل 10 دقائق)
- يمكنك ضغط **"فحص الكل"** للفحص الفوري
- البطاقة تعرض: الحالة، زمن الاستجابة، كود HTTP، آخر فحص
- عند أي تعطّل → تنبيه تلقائي في `/admin/alerts`

## التعطيل المؤقت

اضغط أيقونة ⏻ على بطاقة الموقع → سيتوقف الفحص دون حذف الموقع.

## الحذف

اضغط أيقونة 🗑 → تأكيد → سيُحذف الموقع وكل تاريخه.

> ⚠️ **مهم**: الحذف من مركز التحكم **لا يمسّ** الموقع المنشور أو دومينه.
